# Networking Interview Notes

## Purpose
Covers the networking concepts most likely to appear in cloud architect interviews.

---

## Key Topics
- VNets and subnet strategy
- peering and hybrid connectivity
- private endpoints and DNS considerations
- ingress and egress control
- load balancing and edge security

---

## Sample Architect Answer
I focus on designing secure and simple network flows first. For enterprise Azure solutions, that often means environment separation, private access for sensitive PaaS services, controlled ingress points, and clear hybrid connectivity decisions based on security and operational needs.
