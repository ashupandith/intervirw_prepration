# Security Interview Notes

## Purpose
Interview-focused notes for identity, access, secrets, network security, zero-trust thinking, and secure-by-design architecture.

---

## Core Concepts
- identity-first security
- least privilege access
- managed identities and secret minimization
- zero-trust principles
- encryption, key management, and auditability

---

## Sample Architect Answer
I treat security as a design constraint from the start, not as a post-build review item. In cloud architecture, I prioritize identity, access control, network segmentation where needed, secrets management, observability, and policy enforcement so the system is secure and operable at scale.
