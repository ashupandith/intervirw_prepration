# Senior Architect Interview Prep Kit (15-30 Days)

This workspace is designed for:
- Senior Solution Architect
- Azure Solution Architect
- Cloud & AI Architect

Experience level target: 15+ years, with focus on enterprise architecture, modernization, cloud, AI, and data platforms.

## How to Use This Kit

1. Start with `01-PLAN/30_day_plan.md` for the daily schedule.
2. Use `02-CORE_TOPICS/` for structured technical preparation.
3. Practice from `03-QUESTION_BANK/` every day.
4. Use `04-MOCK_INTERVIEWS/` for timed mock rounds.
5. Build concise speaking points from `05-RESUME_STORYBANK/`.
6. Revise from `06-REVISION/one_page_revision.md` before interviews.

## Suggested Weekly Cadence

- Week 1: Architecture fundamentals + Azure platform depth
- Week 2: Migration, security, integration, and data architecture
- Week 3: AI/GenAI/RAG architecture + scenario design
- Week 4: Mock interviews + targeted weak-area revision

## Daily Execution Pattern (90-150 min)

- 30-45 min: Topic deep dive
- 30-45 min: Q&A speaking practice
- 20-30 min: Case study / whiteboard design
- 10-20 min: Revision notes and action items

## Folder Map

- `01-PLAN/` timeline and checklists
- `02-CORE_TOPICS/` topic notes by domain
- `03-QUESTION_BANK/` technical + leadership questions
- `04-MOCK_INTERVIEWS/` full round simulation scripts
- `05-RESUME_STORYBANK/` project narration and STAR stories
- `06-REVISION/` condensed revision assets
